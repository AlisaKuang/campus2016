Java项目运行说明


1）EffectiveLines――统计一个Java文件的有效行数：
	在IDEA中导入项目，运行EffectLinesRecord.java，在Console中输入Java文件路径，结果在Console中输出

2）ExchangeRate――统计30天内（工作日）人民币汇率中间价（美元、欧元、港币）：
	在IDEA中导入项目，在连接网络的情况下运行ExchangeRate.java，结果Excel文件路径为项目文件夹\result\ExchangeRate.xls

3）CountMostImport――统计指定目录下被import最多的类（不含整个包引入的类）
	在IDEA中导入项目，运行FileRecord.java，在Console中输入指定目录路径，结果在Console中输出
